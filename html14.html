Image Maps

With HTML image maps, you can create clickable areas on an image.

The HTML map tag defines an image map.
An image map is an image with clickable areas.
The areas are defined with one or more area tags.

Create Image Map As well as the Areas


The coordinates for shape="rect" come in pairs, one for the x-axis and one for the y-axis. Two coordinates that form two margins which create a clickable rectangular area and this area becomes clickable and will send the user to the corresponding html page
To add a circle area, first locate the coordinates of the center of the circle then specifc the radius of the circle
The shape="poly" contains several coordinate points, which creates a shape formed with straight lines (a polygon). This can be used to create any shape. By finding the x and y coordinates for all edges The coordinates come in pairs, one for the x-axis and one for the y-axis
