Html Favicon

What is a Favicon - this is a small image displayed next to the page title in the browser tab.



You can use any image you like as your favicon. You can also create your own favicon on sites like http://www.favicon.cc



NOTE: A favicon is a small image, so it should be simple image with high contrast.

HOW TO ADD A FAVICON IN HTML

To add a favicon to your website, either save your favicon image to the root directory of your webserver, or create a folder in the root directory called images, and save your favicon image in this folder. A common name for a favicon image is "favicon.ico".
